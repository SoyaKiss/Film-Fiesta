Film Fiesta


Movie Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Get a full list of movies /movies GET None A JSON object holding data about all movies in array
Get the title of the movie /movies/:Title GET None A JSON object holding all the data about the movie
Get the year the movie was made /movies/:Year GET None A JSON object holding the data about the year made
Get the genre of the movie /movies/:Genre GET None Within an array: Name: string, and Description: string
Get the main actor's name and birthday /movies/:mainActor GET None A JSON Object with Nested Array: Name: string, and Birthday:string
Get the supporting actor's name and birthday /movies/:supportingActor GET None A JSON Object with Nested Array: Name: string, and Birthday:string
Get an Image of the movie /movies/:ImageURL GET None An Image
Get the description of the movie /movies/:Description GET None A JSON object holding data about the description


User Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Allow new users to register /users/:Username POST JSON object a JSON object holding data about the user
Allow new users to add email /users/:Email POST JSON object a JSON object holding Email data
Allow new users to add birthday /users/:Birthday POST JSON object a JSON object holding birthday data
Allow user to add their full names /users/:fullName PUT JSON object a text message saying "Update Success!"
Allow user to deregister /users/:Username DELETE None Text message: (user.name) has been deleted.
Allow user to add a movie to their favlist /users/:favoriteMovies POST None Text message: (movieTitle) has been added to (user.name)'s favorite list!
Allow user to delete a movie from their favlist /users/:favoriteMovies DELETE None Text message: (movieTitle) has been removed from (user.name)'s favorite list.
