Film Fiesta


Movie Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Get a full list of movies /movies GET None a JSON object holding data about all movies in array
Get the title of the movie /movies/:Title GET None a JSON object holding all the data about the movie
Get the genre of the movie /movies/genre/:genre GET None A string; list of all movies under that genre
Get the main actor's name /movies/main/:mainName GET None A string; list of all the movies that this actor is the main actor


User Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Allow new users to register /users POST JSON object a JSON object holding data about the user
Allow user to update their user info /users/:id PUT JSON object a text message saying "Update Success!"
Allow user to deregister /users/:id DELETE None Text message: (user.name) has been deleted.
Allow user to add a movie to their favlist /users/:id/:movieTitle POST None Text message: (movieTitle) has been added to (user.name)'s favorite list!
Allow user to delete a movie from their favlist /users/:id/:movieTitle DELETE None Text message: (movieTitle) has been removed from (user.name)'s favorite list.
