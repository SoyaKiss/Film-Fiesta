Film Fiesta


Movie Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Get a full list of movies /Movies GET None A JSON object holding data about all movies in many arrays
Show the movie by searching by title /Movies/:Title GET None A JSON object holding all the data about that particular movie
Search movie titles by genres /Movies/Genre/:genre GET None Text message: "(movieTitle)'s" are "(genre)"" type movies.
Get the main actor's name /Movies/mainActor/:Name GET None Text message: (mainActorsName) stars in (movieTile)'s // (if more than one)
Get the supporting actor's name /Movies/supportingActor/:Name GET None Text message: (supportingActorsName) is the supporting actor in (movieTile)'s // (if more than one)
Get an Image of the movie /Movies/ImageURL/:movieTitle GET None A http link to the image
Get the description of the movie /Movies/:Title/Description GET None A text that shows the description of the movie


User Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Get a list of all users /Users GET None a JSON object holding arrays for each user
Create a new user /Users POST JSON object a JSON object holding data about the newly created user
Allow registered user to login /Login POST JSON object, Username/Password Params a JSON object holding data about the existing user, JWT generated
Allow user to update info /Users/:Username PUT JSON objects a JSON object holding data about the updated info
User can add movie to favorite movies list /Users/:Username/:movieTitle POST None Text message: (movieTitle) has been added to (Username)'s favorite list!
User can delete a movie from favorite movies list /Users/:Username/:movieTitle DELETE None Text message: (movieTitle) has been removed from (Username)'s favorite list.
Allow user to deregister /Users/:Username DELETE None Text message: (Username) has been deleted.
