Movie Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Get a full list of movies /movies GET None a JSON object holding data about all movies
Get the title of the movie /movies/:Title GET None a JSON object holding data; the movie's title
Get the description about the movie /movies/:Description GET None a JSON object holding data; the movie's description
Get the year the movie was made /movies/:Year GET None a JSON object holding data; the year the movie was made
Get the genre of the movie /movies/:Genre GET None a JSON object holding data; the genre of the movie
Get an image URL of the movie /movies/:ImageURL GET None an image URL
Get the main actor's name, birthday, height /movies/:Main GET None a JSON object; nested array {name: , birthday: , height: ,}

User Endpoints

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Allow new users to register /login POST JSON object; empty fields {name, email,} a JSON object holding data about the user
Allow user to update their user info (username) /login/:username PUT JSON object of users info a text message saying "Update Success!"
Allow user to deregister /login/:username DELETE None a text message saying "Sad to see you go."
Allow user to add a movie to their favlist /login/username/:favlist PUT None a text message saying "Movie Added!"
Allow user to delete a movie from their favlist /login/username/:favlist DELETE None a text message saying "Movie Removed!"
